# Demo

把运行结果的截图和演示视频放到这里，便于提交/验收。

## 建议内容

- `demo/images/`
  - `scene_overview.png`：整体鸟瞰（包含地形、房屋、city、角色、鸟）
  - `city_pbr_closeup.png`：city 建筑近景（能看出 PBR 受光与高光）
  - `characters_and_house.png`：四个角色与房屋区域
- `demo/video/`
  - `demo.mp4`：10~30 秒即可，展示镜头移动、鸟的水平飞行、city 逐步加载出现

## 录制提示（Windows）

- 可用 Xbox Game Bar：`Win + G` → 录制
- 或用 OBS：选择窗口捕获/游戏捕获

> 注意：本仓库无法在 VS Code 自动生成真实截图/视频文件；请在你本机运行后录制并放入此目录。
